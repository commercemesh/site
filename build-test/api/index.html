Commerce Mesh Protocol API (1.0.0)

Download OpenAPI specification:Download

CMP Support: specs@commercemesh.ai URL: https://commercemesh.com License: MIT

The Commerce Mesh Protocol (CMP) API enables discovery and interaction with products across all participating brands in the network. This API provides search, retrieval, and event capabilities for building commerce applications.

Discovery

Product discovery and search endpoints

Search products

Search for products across all brands in the Commerce Mesh network

Authorizations:
bearerAuth
query Parameters
q
string
Example: q=laptop

Search query

org
string
Example: org=urn:cmp:orgid:123e4567-e89b-12d3-a456-426614174000

Filter by organization ID (URN format)

category
string
Example: category=Electronics

Filter by category

minPrice
number <float>
Example: minPrice=10

Minimum price filter

maxPrice
number <float>
Example: maxPrice=1000

Maximum price filter

currency
string
Default: "USD"
Example: currency=USD

Currency for price filters (ISO 4217)

availability
string
Enum: "InStock" "OutOfStock" "PreOrder" "BackOrder"

Filter by availability

page
integer >= 1
Default: 1

Page number

limit
integer [ 1 .. 100 ]
Default: 20

Results per page

sort
string
Default: "relevance"
Enum: "relevance" "price" "name" "date"

Sort field

order
string
Default: "asc"
Enum: "asc" "desc"

Sort order
